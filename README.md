## Portfolio Website
A simple personal portfolio website built using HTML and basic CSS. The website consists of two pages: a Home page and a Contact page. It is designed to provide a clean introduction about me and an easy way for visitors to get in touch.

## Features
Simple and clean user interface
Home page with personal introduction
Contact page for communication details
Responsive layout using basic CSS
Beginner-friendly project structure

## Technologies Used
HTML5
CSS3

## Purpose
This project was created to practice the fundamentals of web development, including page structure, navigation, and styling using HTML and CSS.
